# 🧠 SQL Queries

This folder contains all SQL scripts used to solve business problems in the project.

The queries are organized based on different analytical domains:

- **01_Data_Overview** → Understanding dataset structure (markets, regions, segments)
- **02_Sales_Analysis** → Monthly & yearly sales trends
- **03_Financial_Analysis** → Net sales & P&L insights
- **04_Customer_Market_Analysis** → Top customers and markets
- **05_Product_Analysis** → Product performance & ranking
- **06_Supply_Chain_Analysis** → Forecast accuracy & demand insights
- **07_Contribution_Analysis** → Contribution analysis by categories

Each folder contains optimized SQL queries using concepts like:
- Joins
- Aggregations
- Window Functions
- CTEs (Common Table Expressions)

These queries were used to generate outputs and business insights for the project.

> ⚠️ Note: Dataset is not included due to restrictions. Queries are written based on the original dataset structure.
