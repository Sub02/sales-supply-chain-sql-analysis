# 📊 Output Snapshots

This folder contains the output results generated from SQL queries.

Each subfolder represents a specific business analysis:

- **01_Data_Overview** → Dataset understanding (markets, regions, segments)
- **02_Sales_Analysis** → Monthly & yearly sales trends
- **03_Financial_Analysis** → Revenue insights & P&L analysis
- **04_Customer_Market** → Top customers and markets by sales
- **05_Product_Analysis** → Best-performing products by division
- **06_Supply_Chain** → Forecast accuracy & demand insights

These outputs were used for presentation and business insights.

> ⚠️ Note: Dataset is not shared due to restrictions. Only query outputs are shown.
